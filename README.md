# Request for Proposal: Wireless Infrastructure Deployment 
Company: Willow Middle School 
Author: Josh Fehr

## Organization and Project Background
Willow Middle School is school located in a small town in the lower Manitoba area. The school focuses on grades 4-8, and hosts around 100 students and growing in the expanding Murphtown area. The school building has been around since 1988 and has been slowly modernising for the past couple of years. A part of this modernization process includes a new wireless network. A wireless network would allow staff and students alike to collaborate on their devices easier, as well as increase productivity with online resources. Currently, the school has a temporary wireless network that can't handle the number of users on the network. This issue leads to slow upload and download speeds.

## Project Objectives 
The objectives for this project include: reliable range throughout the building. Support for 500 users across the school, with options for scalability. Strong security and encryption. And monitoring tools to view network activity. 
The wifi range should cover the entire interior of the building, with good signal strength in every room except the gymnasium and lunchroom. We currently only have 100 students, but we want to be prepared for any major growth. We don't want any network disruptions from exterior or interior threat actors, which is why we want strong network security. On top of security, we want control and constant view of network activity.

## Scope of Work 
The project scope will include the following:
- Wireless site survey
- Access point installment
- Integrating wired ethernet into existing infrastructure
- Network testing
- Security configuration
- Training
- Documentaion (including heat map)
- Ongoing Maintenance

## Technical Requirements 
### Mandatory Requirements:
Access points across the building that grants access anywhere in the building with a signal range of -30 to -50 dBm. Weaker signals are acceptable in the gymnasium and student lunchroom, but they can't go lower than -70 bBm. We require Wi-Fi 6 or newer technology to be implemented on our network. The entire network needs to be able to support 500 users concurrently. Each user should get a minimum of 100 Mbps downloading and uploading speeds. The new cabling to be installed has to be Cat 6a or higher quality. Monitoring software/devices will have to be user friendly. The monitoring system should be able to read all network traffic, along with being able to tell where the traffic is coming from geographically.
